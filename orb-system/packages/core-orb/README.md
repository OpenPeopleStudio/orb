# core-orb

Shared types, identity, and configuration for Orb system.

**Orb Role**: Core identity layer (used by all other packages)

## Responsibilities

- Defines `OrbRole` enum (Sol, Te, Mav, Luna)
- Defines `OrbContext` type for role-aware operations
- Provides configuration loading utilities
- Shared types and constants

Implementation in progress. See `docs/ROADMAP.md` for implementation status.
